# Sinister Staycation

Inspired by the Airbnb model, Sinister Staycation offers a platform for renting out haunted apartments.

<h3> Key Features</h3>

- User Roles: Users can register as hosts to rent out properties or as renters looking for accommodations
- Property Listings: Hosts can create and manage detailed listings for their haunted properties, including descriptions, and images
- Reservation System: Renters can easily make reservations for the available haunted properties through a streamlined booking process
- Booking History: Both hosts and renters have access to a comprehensive booking history, displaying past and upcoming reservations
